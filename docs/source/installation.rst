==============
Installation
==============

We recommend creating a conda environment, but this is optional.

++++++++++++++++++
Developer Install
++++++++++++++++++

Execute the following in a terminal session:

.. code-block:: console

  git clone git@github.com:matsengrp/torchdms.git
  cd torchdms
  make install
  make test
